EduChat is a simple real-time messaging web app where students can chat in groups and privately. It is built using HTML, CSS, JavaScript, and Firebase Realtime Database.

Users can:
- Join group chats
- Send private messages
- Set usernames
- Request verification (future feature)
- Chat in real time across devices

This project is hosted using GitHub Pages and powered by Firebase for backend data storage and messaging.

EduChat is designed to help learners communicate, share study help, and build a student community online.
